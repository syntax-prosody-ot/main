SPOT Automated Testing

This is testing for the basic Align and Wrap constraints. All possible option combinations for Align are testing in Test Sets 1 through 10. Wrap is tested in Test Set 11. If a test is failed, a message will appear indicating the options, direction (for align), and the stree/ptree structures being compared.

All tests are based on the comparison between one stree and and a number of different ptrees. The stree has the structure {[a [b [c d]]] [e f]} which contains XPs of the variety:

- maximal, minimal

- maximal, non-minimal

- non-maximal, minimal

- non-maximal, non-minimal

allowing for all options and option combinations to uniquely affect the number of violations.
