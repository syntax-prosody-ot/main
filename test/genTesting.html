Automated GEN Testing

These tests check the number of trees produced by GEN as well as which trees are produced by GEN for three terminal. 14 parameter combinations of GEN are tested. The parameters used are:

- Headedness

- Exhaustivity

- Non-Recursivity

- Non-Unary Phis

- Phi-Wrapped

For more information on SPOT's GEN function and its parameters, see "Candidate Set Cardinalities in SPOT".
