GEN Argument Tests

This testfile checks that standard GEN(), GENwithCliticMovement(), and GENwithPermutation() function correctly when passed certain arguments. The GEN functions can receive an stree, a string of terminals, neither, or both, and these tests make sure that the output is as expected.

WARNING: Tests will fail if pop-up is left onscreen for longer than two seconds. RECOMMENDATION: Use the enter key.
